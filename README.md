# Galileoへのインストール方法

    # cd /home/root/
    # git clone https://github.com/kohei0302/communikerokero.git
    # cd communikerokero/
    # npm install --production
    # npm link mraa

※パスを読み替えて頂ければ、Galileo以外でも動作します。
